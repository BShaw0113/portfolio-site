Tiger Syndicate — Full-Stack Rails Marketplace

Full-stack Ruby on Rails marketplace built for the University of Memphis community, enabling students to buy, sell, and offer services in a secure, role-based platform.

Tiger Syndicate is a deployment-ready Rails application designed around real-world business rules, data integrity, and usability. The platform supports authenticated users, role-based permissions, and robust CRUD workflows.

The project demonstrates end-to-end full-stack development, including database design with PostgreSQL, authentication using Devise and OAuth, automated testing with RSpec, and responsive UI design using Bootstrap.
